**Supported Frameworks:** `React`

**react-i18next** is a popular React localization framework. TanStack i18n handles URL locale, server entry, persist adapters, and `setLocale` — react-i18next handles translation catalogs and rendering.

<Callout type="info">
Full react-i18next integration recipe is planned. Follow the [use-intl guide](/integrations/use-intl) for general concepts on resolving and sync-ing routing with translation providers.
</Callout>
